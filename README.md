# Streamlit 예제 — 분석을 만지는 화면으로

강의 **CH07 · 인터랙티브 앱**에서 다루는 Streamlit 예제입니다.
한 장 HTML 리포트와 달리, 보는 사람이 직접 기간·항목을 바꿔보는 **인터랙티브 대시보드**예요.
이 코드는 모두 Claude Code에게 *"이 데이터로 이런 화면 만들어줘"* 라고 요청해서 만들었습니다.

## 무엇이 들어있나

- `app.py` — 홈 (두 데모 안내)
- `pages/1_캠페인_성과.py` — 채널별 광고 성과 대시보드 (모듈 4 · 캠페인 정산과 같은 데이터)
- `pages/2_묵호_관광트렌드.py` — 한국관광 데이터랩 동해시 방문자·검색 트렌드 탐색기
- `data/` — 위 두 앱이 쓰는 CSV
- `requirements.txt` — 필요한 패키지

## 실행 방법

```bash
# 1. (한 번만) 패키지 설치
pip install -r requirements.txt

# 2. 이 폴더에서 실행 — 브라우저가 자동으로 열립니다
streamlit run app.py
```

브라우저가 안 열리면 터미널에 표시된 `http://localhost:8501` 주소를 직접 엽니다.
사이드바에서 페이지를 고르고, 필터를 움직이면 표와 차트가 즉시 바뀝니다.

## 직접 만들어 보려면

이 폴더를 처음부터 따라 만드는 게 실습입니다. Claude Code에게 이렇게 말하면 됩니다:

> `data/캠페인_성과_2025-2026.csv`로 Streamlit 대시보드를 만들어줘.
> 사이드바에 기간 슬라이더와 채널 선택을 두고, 상단에 총매출·ROAS KPI,
> 그 아래 월별 매출 추이 라인 차트와 채널별 막대 차트를 보여줘.

화면이 마음에 안 들면 *"차트를 막대로 바꿔"*, *"KPI에 전환수도 추가해"* 처럼 말로 고칩니다.
